## Agent skills

### Issue tracker

Issues and PRDs for this repo live as GitHub issues (uses the `gh` CLI). See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical roles kept as-is: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context. Read `CONTEXT.md` first (glossary + ADR index). Then read only the ADRs relevant to your task from `docs/adr/`. Only read `specs/game_engine_spec.md`, `specs/game_design.md`, or `specs/backend_system_design.md` if an ADR doesn't cover your question. See `docs/agents/domain.md`.
